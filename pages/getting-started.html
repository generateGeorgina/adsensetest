[Windows] Install Jest in 5 mins

Before you can start using Jest for testing, you need to set it up in your project. Here are the steps to get started:

  1. Install NodeJS (npm)
  2. Install Jest using npm
  3. Create a test suite.
  4. Write your first test case.
  5. Run your tests with Jest.

1. Install NodeJS (npm)

Go to NodeJS.org and download the installer for Windows. To figure out if your system is 64-bit or 32-bit, go to Properties, then System. Once that has installed, open your Command Prompt and check your npm version by running:

npm -version

2. Install Jest

Open Command Prompt on Windows, navigate to your current project directory. You can use the terminal in VS Code instead, in which case, you should be in the root (the place that contains all the relevant files/folders for your project) of your project already. Run the following command:

npm install --save-dev jest

You should see a new file, package.json in the root of your project directory. Add "type": "module" to your package.json file. This will be within the root curly braces (don't forget the comma). Do the same with "scripts": { "test": "node -- experimental etc. so you can easily run npm run test :

                
{
    "type": "module",
    "scripts": {
            "test": "node ./node_modules/jest/bin/jest.js --runInBand"
        },
    
    "devDependencies": {
        "jest": "^29.7.0"
    }
}

3. Create a test suite

Before you create your test suite, let's create a simple function that we will export called add.

                function add(a, b) {
    return a + b;
}

export { add };

Name your file add.js, then create another file called add.test.js. Import your your function (remember to use the correct path to your file - "./" marks the root of your project).

            import { add } from "./add.js";

describe("Basic maths", () => {
    //test will go here
});

4. Write your first test case

Our first test case will check to see if the function is working properly by passing in 1 and 1 into the function. We expect the answer to be 2.

                import { add } from "./add.js";

describe("Basic maths", () => {
    test("addition", () => {
        expect(add(1, 1)).toBe(2);
    });
});

5. Run your tests with Jest

Type npm run test in your terminal and hit enter. You should see your test suite and all your tests should pass.

npm run test
